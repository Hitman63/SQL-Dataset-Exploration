

## **Online Shopping Analysis Using SQL**

### **Project Overview**

Explore and analyze an online shopping dataset using **SQL** to uncover key insights and understand customer and order patterns.

### **Key Tasks Performed**

* Identify all **unique cities** where customers are located.
* Count the **total number of orders placed in 2017**.
* Calculate the **total sales for each product category**.
* Determine the **percentage of orders paid in installments**.
* Count the **number of customers from each state**.
